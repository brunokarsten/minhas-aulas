Podemos escrever vários tipos de texto

Negrito

Os textos em negritos podem ser escritos assim usando a tag STRONG - (Tag semantica)

Também pode se usar a tag B - (Não semantica)

Itálico/Ênfase

Para transformar para o ITÁLICO usamos a Tag EM (semântica) ou a TAG I (não semântica) = ITÁLICO

Big & Small

A tag BIG ja não está mais em uso porém temos a TAG small que deixa o texto pequeno

Texto deletado

O texto pode até ser lido mas não precisa ser considerado. Usa-se a TAG del - Texto deletado

Texto inserido

É o famoso texto sublinhado. Usa-se a TAG ins - Texto sublinhado

Textos sobrescrito e subcrito

Textos em cima ou embaixo, geralmente números.
Usa-se a TAG sub - H2O para textos abaixos

Usa-se a TAG sup - X2.
